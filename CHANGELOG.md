# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Basic Nuxt 3 project structure
- UI components (Action, Badge, Card, Dialog, Input, Loader, Skeleton, Spinner, Switch, ToastStack)
- Authentication system with session management (login, logout, refresh token)
- Dark mode support with automatic system preference detection
- Toast notification system
- Keyboard shortcuts composable
- Unit tests with Vitest
- ESLint and Prettier configuration
- TailwindCSS setup with PostCSS
- Authentication middleware
- API composable with error handling
- Demo pages (design-system, api-demo, protected)
- Layout with header and footer

### Changed
- Initial stable version of the project
